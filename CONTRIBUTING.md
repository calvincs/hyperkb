## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run the test suite: `.venv/bin/pytest tests/ -v`
5. Submit a pull request

By opening a pull request, you confirm that you have read and agree to the
terms above.
